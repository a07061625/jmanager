# 介绍
基于GO语言的管理工具项目

## 目录介绍
- common: 公共代码
- configs: 配置文件
- crawler: 爬虫
- webhook: 代码自动部署
- proxy: 代理

## 依赖
### 必选
go get github.com/json-iterator/go //json解析
go get github.com/axgle/mahonia //编码转换

### 可选
go get github.com/gocolly/colly //爬虫框架
go get github.com/PuerkitoBio/goquery //爬虫选择器
